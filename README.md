# CakeCraft Studio

CakeCraft Studio is a full-stack web platform designed to help home bakers connect with customers, manage cake orders, and grow their small baking businesses online.

This project was inspired by a real problem in my own life. My mom is a home baker, and I often helped her communicate with customers through direct messages. Customers would ask about cake flavors, sizes, prices, pickup times, custom designs, and availability, all through scattered conversations. It became clear that home bakers need a more organized way to manage their business, and customers need an easier way to browse, customize, and order cakes.

CakeCraft Studio was built to turn that everyday challenge into a real digital solution.

## Why This Project Was Built

Many home bakers run their businesses through Instagram, WhatsApp, or other messaging apps. While that works at the beginning, it can quickly become overwhelming when orders, customer details, pricing, images, and deadlines are all handled manually.

CakeCraft Studio solves this by creating one central platform where:

- Customers can discover local home bakers.
- Bakers can showcase their cakes and business information.
- Customers can order ready-made cakes or request custom cakes.
- Bakers can manage orders, update statuses, and track upcoming deadlines.
- Both sides can communicate expectations more clearly.

The goal of this project is to support small home-based baking businesses and make the ordering experience smoother, more professional, and easier to manage.

## Design

The user interface and user experience for CakeCraft Studio were designed in Figma before development. The design process helped define the platform’s layout, visual style, user flows, and separate experiences for customers and bakers.

## Features

### Customer Features

- Customer sign up and sign in
- Browse local bakers
- View baker profiles
- Search for cakes, flavors, and bakers
- Save favorite bakers or cakes
- Order ready-made cakes
- Build a custom cake request
- Add delivery or pickup details
- Manage payment methods
- Track current and past orders
- Leave ratings and reviews for completed orders

### Baker Features

- Baker sign up and sign in
- Dashboard for managing business activity
- Create, edit, and manage cake listings
- Upload cake images
- Set cake details such as flavor, size, shape, fillings, toppings, and price
- Manage incoming orders
- Accept, decline, or adjust order requests
- View orders in a calendar
- Track completed orders
- Edit bakery profile and business settings
- Manage payout information
- Receive email notifications for important order updates

## What I Learned

This project helped me understand how to build a real full-stack application based on an actual user problem. I worked with UI/UX design in Figma, frontend routing, backend APIs, database models, authentication, image uploads, order workflows, payment-related UI, and email notifications.

More importantly, I learned how software can support small businesses by reducing manual work and making everyday tasks easier to manage.
